#Setup

##Check repo out
	git clone https://github.com/nicam/threejs-macbook

##Run Webserver
	cd threejs-macbook
	python -m SimpleHTTPServer

##Enjoy
Go to <a href="http://localhost:8000">http://localhost:8000</a>
